Every trend has a window. The pipeline finds it. The 48-hour rule says: if you can't validate it and ship something useful in 48 hours, the window has already moved.

That's the whole operating theory.

## How the decision happens

Trend-detector runs at 9AM. It pulls from three sources — real-time Google trends, the public RSS feed, and a curated Gmail digest. Cross-references everything. Scores each keyword across four axes: search volume trajectory, competitor density, Reddit pain signal, and where the trend sits in its lifecycle curve.

BUILD GREEN means early stage, low competition, confirmed pain. Those are the only ones worth 48 hours.

Most signals get WATCH or SKIP. Watching costs nothing. Skipping is the point — the filter is more valuable than the output.

## Why 48 hours

Not because speed is a virtue. Because trend windows close.

When something scores BUILD GREEN, there are no tools for it yet. That's what makes it worth building. Two weeks later there are three. The window was open for ten days and I missed it.

The 48-hour constraint also forces a real scope decision. Not five features. One. What is the one thing this tool needs to do? The window is short enough that scope creep is physically impossible.

## What compounds

Each tool is a tool. But each tool also builds three things that outlast it.

**Domain.** Everything ships under modrynstudio.com. Domain authority compounds across every tool. When the twelfth tool ships, it inherits twelve tools' worth of trust signals. Separate domains don't do this.

**Email list.** One list, segmented per tool. Every signup on any tool feeds the same base. When the next tool drops, I already have an audience that opted in to exactly this kind of thing.

**Content.** The build log is the distribution. Every post about the process — what the pipeline found, what I built, what worked — is content that can be cross-posted, shared, linked back. Building in public is not optional. It's the strategy.

## Where this stands

The pipeline is solid. The decision framework is solid. I have not yet taken a BUILD GREEN signal, built the thing in 48 hours, and watched the traffic arrive.

That's the test. Everything until that happens is setup.

The test is next.
